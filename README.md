## About the Project

**Chickengram** is a fork of the official Telegram client for Android, created for customization, experimentation, and feature expansion.  
The project is based on the original **Telegram Android source code** from the **[DrKLO/Telegram repository](https://github.com/DrKLO/Telegram)**  
and enhanced with modern development tools and CI/CD.
